Introduction
============

|GNATformat| is a an opinionated code formatter for the Ada language.

|GNATformat| fits naturally into a software development team's workflow
by leveraging on project files to format a valid Ada source code according
to the coding style described in the |GNAT-Style| available online at
https://gcc.gnu.org/onlinedocs/gnat-style.pdf.

|GNATformat| is available as a command line tool or as a library on the
following platforms

* Windows (64-bits)
* Linux (64-bits)


Prerequisites
-------------

In order to use |GNATformat| you must have |GNATPro| installed and available
in your :envvar:`$PATH`.
